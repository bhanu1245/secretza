CRITICAL DEFECT

Issues found

  • Location-pin shape: The shield path tapers to a point at y=58. At small sizes it reads as a map/location marker with no brand identity.
  • SZ letters invisible at favicon sizes: The SZ strokes are 2.35px on a 64-unit canvas. At 16px render, they disappear completely.
  • Light-theme silhouette defect: logo-icon-light.svg uses fill="#FFFFFF" for the silhouette cutout — it becomes invisible against light backgrounds.
  • Checkmark unrecognizable: The verification circle at cy=48.5 is at the bottom tip of the shield. Invisible below 48px render, adds visual noise above it.
  • Overcrowded mark: Shield + SZ monogram + feminine silhouette cutout + checkmark = 4 competing elements in 64×64 pixels. Luxury marks are minimal.
  • Silhouette barely recognizable: The "feminine silhouette" is a narrow 10px-wide negative-space cutout. At any size below 128px, it reads as a random squiggle.
  • Typography not luxury: BrandWordmark uses default font-bold (700) without letter-spacing. Luxury wordmarks use 800 weight + wide tracking.
  • Component architecture is good: Single LogoMark.tsx source of truth, BRAND_ASSETS paths used consistently, no stray hardcoded paths.
